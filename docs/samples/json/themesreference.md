## Formatos json de temas con sus variantes de color

# theme-win95:

```json
  "settings": {
        "accentColor": "#c0c0c0",
        "colors": {
          "bg": "#3a3a3a",
          "primary": "#2b2b2b",
          "score": "#ffffff",
          "secondary": "#8c8c8c",
          "text": "#000000",
          "window": "#6b6b6b"
        },
        "customTitle": "Liga S Potosina - Semana 10",
        "opacity": 100,
        "orientation": "vertical",
        "showPhotos": true,
        "swapPlayers": true,
        "theme": "theme-win95",
        "variant": "gold", // Esto no debería de estar aquí o de menos debería poderse modificar cada que se elige un tema diferente y su variante
        "verticalMode": false
      }
```
